Module Plateau¶

plateau.correspond(x, y)¶
Associe une valeur en pixel x,y à une case du plateau plat
Parameters:
  • x (entier) – valeur x en pixel (abscisse)
  • y (entier) – valeur y en pixel (abscisse)
Return type:

tuple
Returns:

valeur des cases i et j

>>>correspond(350,850) (5,4)

plateau.couleur_case(i, j)¶
Détermine la couleur d’une case dans le plateau au coordonnées (i,j)
Parameters:
  • i (entier) – coordonnée i correspond à l’abscisse d’une case
  • j (entier) – coordonnée j correspond à l’ordonnée d’une case
Return type:

booléen
Returns:

True pour la couleur blanc, False pour la couleur noire
plateau.dans_plateau(x, y)¶
Verifie si des coordonnées x,y sont bien comprises dans le plateau de jeu
Parameters:
  • x (entier) – coordonnée x
  • y (entier) – coordonnée y
Return type:

booléen
Returns:

True dans le plateau sinon False en dehors
